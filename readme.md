## 部分地区电视台网络源

## 使用说明

1. 运行部署
```
pip install flask
python main.py
```

2. 使用 OTT 软件或各类视频播放器打开下列链接即可
```
# 怀化综合频道
http://ip:10492/zonghe

# 怀化公共频道
http://ip:10492/gonggong

# Now 新闻频道
http://ip:10492/now_news

# Now 财经频道
http://ip:10492/now_business

# Viu
http://ip:10492/viu

# Viu 6
http://ip:10492/viu_six

# HBO HK
http://ip:10492/hbo_hk
```
